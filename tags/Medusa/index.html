    Speculative Decoding & Continuous Batching: LLM Inference Layer Architecture 완전 분석 (#052)

    2026년 현재 LLM inference latency의 73%는 GPU가 idle한 시간이다. 원인: autoregressive generation의 본질적 순차성 + 정적 batching의 padding waste. 이 글에서는 Speculative Decoding(Medusa, EAGLE, Lookahead), Continuous Batching(vLLM, SGLang의 RadixAttention), 그리고 PagedAttention의 KV cache 메모리 엔지니어링을 production 코드와 함께 완전 분석한다. 코드 한 줄이 throughput을 23배 바꾸는 세계.
