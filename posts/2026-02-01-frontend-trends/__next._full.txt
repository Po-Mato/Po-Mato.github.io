<h3>1. 🤖 transformers.js (Hugging Face)</h3>
<p><strong>GitHub:</strong> <a href="https://github.com/huggingface/transformers.js">https://github.com/huggingface/transformers.js</a></p>
<h4>뭐하는 프로젝트?</h4>
<ul>
<li>브라우저에서 서버 없이 Hugging Face의 Transformer 모델을 직접 실행할 수 있는 JavaScript 라이브러리</li>
<li>ONNX Runtime을 활용해 클라이언트 사이드에서 ML 추론 가능</li>
</ul>
<h4>왜 트렌드에 올랐는지?</h4>
<ul>
<li>AI 붐과 맞물려 프론트엔드에서 직접 ML 모델을 돌릴 수 있다는 점이 큰 화제</li>
<li>서버 비용 절감과 개인정보 보호 측면에서 강력한 이점 제공</li>
<li>WebGPU 지원으로 성능도 크게 향상</li>
</ul>
<h4>실무 체크리스트</h4>
<ul class="contains-task-list">
<li class="task-list-item"><input type="checkbox" disabled> 서버 없이 텍스트 분류/요약/번역 기능을 프론트엔드에 구현 가능한지 검토</li>
<li class="task-list-item"><input type="checkbox" disabled> 사용자 데이터를 서버로 보내지 않고 처리해야 하는 프라이버시 민감 기능에 활용</li>
<li class="task-list-item"><input type="checkbox" disabled> 오프라인 환경에서도 동작하는 AI 기반 기능 구현 시 유용</li>
</ul>

<h3>2. 📱 whatsapp-web.js</h3>
<p><strong>GitHub:</strong> <a href="https://github.com/pedroslopez/whatsapp-web.js">https://github.com/pedroslopez/whatsapp-web.js</a></p>
<h4>뭐하는 프로젝트?</h4>
<ul>
<li>Puppeteer를 통해 WhatsApp Web을 제어하는 Node.js 클라이언트 라이브러리</li>
<li>봇 개발, 자동화, 알림 시스템 구축에 활용 가능</li>
</ul>
<h4>왜 트렌드에 올랐는지?</h4>
<ul>
<li>WhatsApp Business API의 높은 진입 장벽 때문에 커뮤니티 기반 대안 솔루션 수요 증가</li>
<li>Multi-Device 지원, 미디어 전송, 그룹 관리 등 거의 모든 WhatsApp 기능 커버</li>
<li>고객 서비스 자동화, 알림 시스템에 대한 실무 니즈 반영</li>
</ul>
<h4>실무 체크리스트</h4>
<ul class="contains-task-list">
<li class="task-list-item"><input type="checkbox" disabled> 고객 문의 자동 응답 봇 구현 시 WhatsApp 채널 추가 검토</li>
<li class="task-list-item"><input type="checkbox" disabled> 주문/배송 알림을 WhatsApp으로 전송하는 시스템 구축 가능성 탐색</li>
<li class="task-list-item"><input type="checkbox" disabled> ⚠️ 공식 API가 아니므로 계정 차단 위험 인지 필요 (프로덕션 환경 주의)</li>
</ul>

<p><strong>GitHub:</strong> <a href="https://github.com/crocodilestick/Calibre-Web-Automated">https://github.com/crocodilestick/Calibre-Web-Automated</a></p>
<h4>뭐하는 프로젝트?</h4>

<li>Calibre-Web에 자동화 기능을 대폭 추가한 eBook 관리 솔루션</li>

<li>Calibre의 복잡한 UI 대신 모던한 웹 인터페이스 제공</li>
<li>OAuth 2.0/OIDC 인증, 배치 편집, 통계 대시보드 등 엔터프라이즈급 기능 탑재</li>
</ul>

<li class="task-list-item"><input type="checkbox" disabled> 파일 업로드 → 자동 변환 → 저장 워크플로우를 다른 도메인에 적용 가능</li>
